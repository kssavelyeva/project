Проект RandomPeopleGenerator

Запуск проекта.
1. Запустить командную строку от имени администратора.
2. Перейти в папку с проектом при помощи комманды cd + путь. Пример: cd C:\Users\Карина\IdeaProjects\RandomPeopleGenerator
3. Выполнить команду: mvn compile
4. Выполнить команду: mvn exec:java -Dexec.mainClass="Generator"
5. В папке resources (путь:src\main\resources) создались два файла "Files" форматов: pdf и xls.
